  • 100% battery status Light up the LEDs one by one to simulate current flowing in the battery

    75% - 100% battery status Light up the LEDs one by one to simulate current flowing in the battery. Blink with the last one.

    50% - 75% battery status Light up the LEDs one by one to simulate current flowing in the battery. Blink with the last one.

    25% - 50% battery status Light up the LEDs one by one to simulate current flowing in the battery. Blink with the last one.

    5% - 25% battery status Light up the LEDs one by one to simulate current flowing in the battery. Blink with the last one.

    0 - 5% battery status Blink with the first LED to show that you need to charge your Memoto soon

    Power on Start at 0 and fade all LEDs to 100 to simulate that it is waking up

    Power off Start at 100 and fade all LEDs to 0 to simulate that it is going to sleep (in a pocket or laying flat)

    Double tapped 4,3,2,1 Smile! Photo is taken when all LEDs light up at the end

    Out of storage Fade from 0 to 100 and back again to show that it is soon out of storage. This is by purpuse simular to the "out of battery" state because the end result for the user is that he needs to connect it to a computer as soon as possible.

    Charging When connected to a computer or other power. Light up all LEDs and then act as tapped.

    Plugged out Blink once to indicate that it working
